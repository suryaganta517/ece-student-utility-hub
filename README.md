# ECE Student Utility Hub

ECE Student Utility Hub is a simple web-based platform created to help Electronics and Communication Engineering students quickly use common academic and engineering tools.

## Features

- Binary to Decimal Converter
- Decimal to Binary Converter
- CGPA Calculator
- Antenna Wavelength Calculator
- Resistor Color Code Calculator
- Student-friendly interface
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript

## Live Demo

[Visit Website](https://ece-student-tools.netlify.app)

## Purpose

This project was created to provide useful tools for ECE students in one place and to practice frontend web development skills.
